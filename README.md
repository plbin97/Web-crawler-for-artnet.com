# Web crawler for artnet.com
This program would catch all the data from https://www.artnet.com/auctions/ for scientific research and big data analysis

# Environment Required
This program required python 2.0+ but less than python 3.0
Modules required: requests, Regular Expression, datetime, MySQLdb(If you need)

# How to use: 
First, you have to edit config.py to configurate the program before you use. 
Then, execute run.py
